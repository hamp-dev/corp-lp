# hamp カタログ作成ガイド：スクショ差し込み指定

## 撮影ルール

- ブラウザのURL、ブックマーク、通知、Slack、開発者ツール、個人情報は写さない。
- 画面幅はできれば1440px前後、本文の表示倍率は100%。
- 画像形式はPNGまたはWebP推奨。
- 1枚のスクショで説明したいボタン・入力欄が見えるように撮る。
- 余白が広すぎる場合は、画面全体ではなく該当領域を少し広めに切り抜く。
- 文字が小さくなる場合は、LP側で横幅いっぱいに表示する前提で撮る。
- ダミーのイベント名・サークル名・メールアドレスを使う。
